# Regional Operational Spend & Corporate Cost Control Framework

## 📌 Business Case & Problem
Fragmented operational expense logs and manual tracking bottlenecks left the management team blind to daily cash outlays.

## 🛠️ Technical Solution & Tooling
* **Core Stack:** Microsoft Excel (Advanced Dashboards, Formulas & Pivot Tables)
* **The Fix:** Engineered a dynamic, interactive Excel tracking dashboard to centralize, filter, and audit cross-category regional expenses.
